# Actividad 8 - CRUD Superheroes (Laravel 7)

## 📌 Descripción
Sistema CRUD desarrollado en Laravel 7 que permite gestionar superhéroes.

## 🚀 Funcionalidades
- Crear superhéroes
- Ver superhéroes
- Editar superhéroes
- Eliminar superhéroes
- Listar todos los registros

## 🧩 Campos
- Nombre real
- Nombre de superhéroe
- Foto (URL)
- Descripción

## ⚙️ Tecnologías
- Laravel 7
- MySQL
- Laragon
